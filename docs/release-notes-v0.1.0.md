# Codex Titlebar Meter v0.1.0

首个正式可用版本。无需打开账户菜单，直接在 Codex Desktop 标题栏查看真实剩余额度。

## 中文

- 显示额度周期、剩余百分比、重置时间和进度条。
- 紧凑靠右，不遮挡 Codex 的“文件 / 编辑”等菜单。
- 自动跟随 Codex 窗口，最小化或退出 Codex 时自动隐藏或退出数据连接。
- 当前支持中文和英文；其他 Codex 界面语言暂时回退为英文。
- 无需 API Key，不复制登录凭据，不修改 Codex 文件，没有遥测。
- 支持便携运行、当前用户自动启动以及 Windows“已安装的应用”卸载。

## English

The first production-ready release. See real Codex quota information directly in the Codex Desktop title bar without opening the account menu.

- Shows the quota window, remaining percentage, reset time, and a progress bar.
- Uses a compact right-aligned layout that does not cover Codex menus.
- Follows the Codex window and pauses its data connection when Codex is minimized or closed.
- Supports Chinese and English; other Codex UI languages currently fall back to English.
- Requires no API key, copies no login credentials, modifies no Codex files, and sends no telemetry.
- Supports portable use, per-user automatic startup, and clean uninstall through Windows Settings.
