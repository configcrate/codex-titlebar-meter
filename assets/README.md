# Visual assets / 宣传素材

所有宣传图同时保留 SVG 源稿和 PNG 成品。修改文字、百分比或日期时编辑 SVG，再重新渲染 PNG。

| 文件 | 尺寸 | 用途 |
| --- | ---: | --- |
| `github-hero-zh.png` | 1280×640 | GitHub README 中文主图、中文文章正文配图 |
| `github-hero-en.png` | 1280×640 | GitHub README 英文主图、GitHub 社交预览 |
| `titlebar-detail-zh.png` | 1280×420 | 功能位置详解、文章正文配图 |
| `wechat-cover-zh.png` | 900×383 | 微信公众号头条封面 |

图中界面为经过简化的产品效果示意，重点准确展示标题栏用量条的位置和信息结构，不包含用户数据。
